# Intention
I first tried to do this with the knowledge of a basic Python course and the help of Automate The Boring Stuff with Python. It was really messy. Now armed with 60-day Boot.dev streak, 5 modules  and 2 guided projects completed, I'm going to try again.

Indie Hackers (IH) is a podcast that is no longer active as of now, with the last episode (#283) published on 15 June 2023. It's well loved including by me. One of the greatest value I got listening to IH is getting to know actual people and their actual businesses, and then finding them online. Thus the idea to scrape the podcast for all the names and links of the people interviewed.

I hope that it can also be used for other interview-based podcasts. Let's go!
